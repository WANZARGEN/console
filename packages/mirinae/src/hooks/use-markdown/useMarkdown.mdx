# useMarkdown


The `useMarkdown` hook parses markdown text and sanitizes it for safe HTML rendering.

## Type Declarations

```typescript
interface UseMarkdownOptions {
    value: Ref<string>|string; // Markdown content to display. string or Ref<string>
}
interface UseMarkdownReturn {
    sanitizedHtml: Ref<string>; // Sanitized HTML content
}
```
